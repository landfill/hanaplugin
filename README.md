# hanaplugin

Cursor 팀 마켓플레이스용 플러그인 모음.

## 포함 플러그인

- **product-research** — 프로덕트 시장조사 & 기획 프롬프트 빌더

## Import 방법

Cursor Dashboard → Settings → Plugins → Team Marketplaces → Import 후 아래 URL 입력:

```
https://github.com/landfill/hanaplugin
```

상세 사용법은 [plugins/product-research/README.md](plugins/product-research/README.md) 참고.
